# Phoenix APS — Windows Installer

**Advanced Planning & Scheduling** — finite-capacity, multi-resource scheduling engine with calendars, precedences, setups, and material awareness.

[www.phoenixaps.com](https://www.phoenixaps.com)

## Download

Get the latest installer from [Releases](https://github.com/phoenixaps-solution/phoenix-aps-installer/releases/tag/latest).

## What's included

- Angular 19 frontend (SPA) served via nginx
- Phoenix.APS.API — core scheduling engine (.NET 9)
- Phoenix.UserManagement.API — authentication & user management
- Nginx reverse proxy with self-signed TLS certificate
- SQL Server database backups (Starter + Demo)

## Quick start

1. Download and run `setup-phoenix-aps.exe` as Administrator
2. Restore the database backups → see [Database Installation Guide](help/documentation/Phoenix_APS_Database_Installation_Guide.pdf)

## About

Phoenix APS is a personal, non-commercial R&D project currently in beta. This repository publishes the **Windows installer only** — no source code is hosted here. The installer bundles all necessary components for on-premise deployment.
